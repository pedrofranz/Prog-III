Programação 3 – Java

Este repositório tem como finalidade armazenar e organizar os exercícios e atividades desenvolvidos na disciplina de Programação 3, utilizando a linguagem Java. O conteúdo aqui presente inclui tanto as atividades realizadas em sala de aula quanto os exercícios extraclasse (ETX), com o intuito de registrar o processo de aprendizado e o desenvolvimento das habilidades de programação orientada a objetos.

A disciplina faz parte da grade curricular do curso de Ciência da Computação da Universidade Federal de Mato Grosso (UFMT), Campus Universitário de Barra do Garças, e tem como foco o aprimoramento das técnicas de programação, o uso de boas práticas de desenvolvimento e a consolidação dos fundamentos da linguagem Java.
